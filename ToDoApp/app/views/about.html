About

A basic sample app. I've primarily used this as a test bed to get familiar with working with Git and Github.

Planning to continue to iterate and utilise the basic foundations of the app for experimenting and testing new technology and frameworks.

Technologies used:
  • AngularJS
  • Typescript
  • Git DVCS
  • Firebase (backend as a service)
  • GitHub and GithubPages
  • Toaster JS library
